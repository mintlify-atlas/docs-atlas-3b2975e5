---
title: Snapshots
description: Manage and restore backup snapshots
---

Snapshots are point-in-time backups stored in repositories. Each snapshot contains files from a backup job and includes metadata like tags, duration, and retention categories.

## List Snapshots

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:4096/api/v1/repositories/{shortId}/snapshots?backupId=abc123" \
  -H "Cookie: zerobyte.session=..."
```
</CodeGroup>

List all snapshots in a repository, optionally filtered by backup schedule.

**Parameters:**

<ParamField path="shortId" type="string" required>
  Repository short ID
</ParamField>

<ParamField query="backupId" type="string">
  Filter by backup schedule short ID
</ParamField>

**Response:**

<ResponseField name="snapshots" type="array">
  <Expandable>
    <ResponseField name="short_id" type="string">
      Snapshot short ID
    </ResponseField>
    <ResponseField name="time" type="number">
      Snapshot creation timestamp (milliseconds)
    </ResponseField>
    <ResponseField name="paths" type="string[]">
      Backed up paths
    </ResponseField>
    <ResponseField name="size" type="number">
      Total bytes processed
    </ResponseField>
    <ResponseField name="duration" type="number">
      Backup duration in seconds
    </ResponseField>
    <ResponseField name="tags" type="string[]">
      Snapshot tags
    </ResponseField>
    <ResponseField name="retentionCategories" type="string[]">
      Retention categories (e.g., "daily", "weekly")
    </ResponseField>
    <ResponseField name="hostname" type="string">
      Hostname where backup was created
    </ResponseField>
    <ResponseField name="summary" type="object">
      Restic backup summary with detailed statistics
    </ResponseField>
  </Expandable>
</ResponseField>

## Download Snapshot Files

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:4096/api/v1/repositories/{shortId}/snapshots/{snapshotId}/dump?path=/data/file.txt&kind=file" \
  -H "Cookie: zerobyte.session=..." \
  -o downloaded-file.txt
```
</CodeGroup>

Download files or directories from a snapshot as a stream.

**Parameters:**

<ParamField path="shortId" type="string" required>
  Repository short ID
</ParamField>

<ParamField path="snapshotId" type="string" required>
  Snapshot ID
</ParamField>

<ParamField query="path" type="string">
  File or directory path to download
</ParamField>

<ParamField query="kind" type="string">
  Path type: "file" (raw stream) or "dir" (tar archive)
</ParamField>

**Response:**

- **For files:** Raw file stream with `Content-Type: application/octet-stream`
- **For directories:** Tar archive with `Content-Type: application/x-tar`

The `Content-Disposition` header includes the appropriate filename.

## Restore Snapshot

<CodeGroup>
```bash cURL
curl -X POST http://localhost:4096/api/v1/repositories/{shortId}/restore \
  -H "Cookie: zerobyte.session=..." \
  -H "Content-Type: application/json" \
  -d '{
    "snapshotId": "abc12345",
    "targetPath": "/mnt/restore",
    "include": ["/data/*"],
    "exclude": ["*.tmp"],
    "overwrite": "always"
  }'
```
</CodeGroup>

Restore a snapshot to a target path on the filesystem.

**Parameters:**

<ParamField path="shortId" type="string" required>
  Repository short ID
</ParamField>

**Request Body:**

<ParamField body="snapshotId" type="string" required>
  Snapshot ID to restore
</ParamField>

<ParamField body="targetPath" type="string">
  Target filesystem path (defaults to original location)
</ParamField>

<ParamField body="include" type="string[]">
  Patterns to include (glob patterns)
</ParamField>

<ParamField body="exclude" type="string[]">
  Patterns to exclude (glob patterns)
</ParamField>

<ParamField body="excludeXattr" type="string[]">
  Extended attributes to exclude
</ParamField>

<ParamField body="delete" type="boolean">
  Delete extra files not in snapshot
</ParamField>

<ParamField body="overwrite" type="string">
  Overwrite mode: "always", "never", or "if-newer"
</ParamField>

**Response:**

<ResponseField name="success" type="boolean">
  Whether restore succeeded
</ResponseField>

<ResponseField name="message" type="string">
  Result message
</ResponseField>

<ResponseField name="filesRestored" type="number">
  Number of files restored
</ResponseField>

<ResponseField name="filesSkipped" type="number">
  Number of files skipped
</ResponseField>

## Tag Snapshots

<CodeGroup>
```bash cURL
curl -X POST http://localhost:4096/api/v1/repositories/{shortId}/snapshots/tag \
  -H "Cookie: zerobyte.session=..." \
  -H "Content-Type: application/json" \
  -d '{
    "snapshotIds": ["abc123", "def456"],
    "add": ["production", "important"],
    "remove": ["test"]
  }'
```
</CodeGroup>

Add, remove, or set tags on multiple snapshots.

**Parameters:**

<ParamField path="shortId" type="string" required>
  Repository short ID
</ParamField>

**Request Body:**

<ParamField body="snapshotIds" type="string[]" required>
  Array of snapshot IDs (minimum 1)
</ParamField>

<ParamField body="add" type="string[]">
  Tags to add
</ParamField>

<ParamField body="remove" type="string[]">
  Tags to remove
</ParamField>

<ParamField body="set" type="string[]">
  Replace all tags with this list
</ParamField>

**Response:**

<ResponseField name="message" type="string">
  Success message
</ResponseField>
